# AESTHETICA — AI-Assisted Architecture & Interior Design Landing Page

## Overview

AESTHETICA is a modern responsive landing page concept for an architecture and interior design studio, created with a next-generation development workflow that combines frontend engineering with AI-assisted design and productivity tools.

The project focuses on premium minimalist UI, smooth user experience, interactive client-side functionality, and rapid modern development practices powered by AI-enhanced workflows.

The website was built as a fully responsive single-page application using semantic HTML, TailwindCSS, vanilla JavaScript, and AI-assisted ideation/refinement during the development process.

---

## Features

### Responsive Design

* Fully adaptive layout for desktop, tablet, and mobile devices
* Mobile navigation menu with animated hamburger transition
* Smooth scrolling navigation
* Mobile-first responsive structure

### Modern UI/UX

* Minimalist luxury-inspired visual direction
* AI-assisted visual refinement and layout iteration

* Soft neutral color palette

* Clean spacing and premium visual hierarchy

### Interactive Components

* Dynamic project filtering system
* Real-time project cost calculator
* Interactive range slider
